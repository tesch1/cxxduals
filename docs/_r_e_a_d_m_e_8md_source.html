1 ## cxxduals
2 
3 *Under Construction!*
4 
5 Template header library for [dual
6 numbers](http://en.wikipedia.org/wiki/Dual_number) in C++. Dual
7 numbers are related to complex numbers, but &epsilon; is nilpotent
8 (&epsilon; * &epsilon; = 0). Nesting provides hyperdual numbers, and
9 hyperhyperdual numbers, etc...
10 
11 Also compiles under CUDA.
12 
13 [![Build Status](https://api.travis-ci.org/tesch1/cxxduals.svg?branch=master)](http://travis-ci.org/tesch1/cxxduals)
14 [![Coverage Status](https://coveralls.io/repos/github/tesch1/cxxduals/badge.svg?branch=master)](https://coveralls.io/github/tesch1/cxxduals?branch=master)
15 
16 Using dual numbers in place of basic types provides a simple way to
17 compute the derivative of a function, because:
18 
19 \f$ f(x + &epsilon; y) = f(x) + &epsilon; f'(x) y \f$
20 
21 Thus, to calculate \f$ f'(3) \f$, set \f$ y = 1 \f$, \f$ z = (3 +
22 &epsilon; 1) \f$ and take the \f$ &epsilon;\f$-part of \f$ f(z) \f$ =>
23 \f$ epart(f(z)) := f'(3) \f$
24 
25 Shortcuts into the docs:
26 
27 - `namespace cxxduals`
28 - `cxxduals::dual`
29 - `cxxduals::sin`

31 ## Usage
32 
33 A simple example calculating (x*x) and the derivative of (x*x):
34 
35 ~~~~~~~~~~~~~~~{.cpp}
36 
37  #include <iostream>
38  #include <cxxduals/dual>
39  using namespace cxxduals;
40 
41  int main(int argc, char *argv[])
42  {
43  dual<double> x;
44  x = {3,1};
45 
46  std::cout << "x=" << x << "\n";
47  std::cout << "x*x=" << x*x << "\n";
48  std::cout << "epart(x*x) = d(x*x)/dx = " << epart(x*x) << "\n";
49  }
50 
51 ~~~~~~~~~~~~~~~
52 
53 Produces:
54 
55 ~~~~~~~~~~~~~~~
56  x=(3 + e1*1)
57  x*x=(9 + e1*6)
58  epart(x*x) = d(x*x)/dx = 6
59 ~~~~~~~~~~~~~~~
60 
61 
62 Before including the header, some "#defines" will modify what's done by
63 the header:
64 
65 ~~~~~~~~~~~~~~~{.cpp}
66 
67 #define CXXDUALS_NO_LIMITS // disable specialization of std::numeric_limits<>
68 #define CXXDUALS_NO_TYPEDEFS // disable typedefs for `dualf,duald,dualld,dualcf,dualcd,dualcld`
69 #define CXXDUALS_EIGEN // force inclusion of Eigen interface (normally auto-detected)
70 #include <cxxduals/dual>
71 
72 ~~~~~~~~~~~~~~~

95 ## History
96 
97 Although it has diverged significantly, this started as a dual-number
98 class implementation by Jeff A. Fike found here:
99 [http://adl.stanford.edu/hyperdual/](http://adl.stanford.edu/hyperdual/).
100 
101 It's been generalized for nesting, cleaned up for easier use in C++,
102 and made suitable for use in
103 [Eigen](http://eigen.tuxfamily.org/index.php?title=Main_Page)
104 matrices.
